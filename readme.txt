Quick Start Instructions

a) Install Python 3.x

b) Open a Terminal and navigate to this file's download directory.

c) In the terminal, run the command "pip install -r requirements.txt" to install dependencies. 

c) In the terminal, run the command "python main.py" to start the inventory conversion tool.

e) Open a browser and enter the address: http://127.0.0.1:5000

f) Operate the conversion calculator as per the instructions in the original paper 'Linking Symptom Inventories using Semantic Textual Similarity'
